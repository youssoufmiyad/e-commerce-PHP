# E-commerce PHP

## Répartition des tâches : 
- Front-End :
    - design des différentes pages (authentification, profil, panier, etc...)
    - routage
    - gestion de l'authentification de l'utilisateur

- Back-End :
    - création/implémentation de la BDD
    - dashboard pour les administrateurs :
        - CRUD produits
        - CRUD utilisateurs
